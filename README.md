# Mobile Shop Management System (C# WinForms + MS Access) 📱🛒

โปรเจกต์นี้เป็นระบบบริหารจัดการร้านขายโทรศัพท์มือถือแบบครบวงจร พัฒนาด้วยภาษา **C#** บนเฟรมเวิร์ก **Windows Forms** โดยมีการเชื่อมต่อกับฐานข้อมูล **Microsoft Access** เพื่อจัดการข้อมูลสินค้า สต็อก และการขายสินค้าหน้าบัญชี

---

## 🔍 ภาพรวมของโปรเจกต์

ระบบถูกออกแบบมาเพื่อให้เจ้าของร้านสามารถจัดการหน้าร้านได้อย่างมีประสิทธิภาพ รองรับการทำงานตั้งแต่การจัดการแคตตาล็อกสินค้า (โทรศัพท์มือถือแบรนด์ต่างๆ) การควบคุมสต็อกสินค้าเข้า-ออก และระบบจัดการข้อมูลพนักงาน โดยเน้นการใช้งานที่ง่ายผ่าน GUI ที่แบ่งสัดส่วนฟังก์ชันไว้อย่างชัดเจน

โปรเจกต์นี้เหมาะสำหรับศึกษาเรื่อง **Database Connectivity (OLEDB)**, **CRUD Operations** ใน C# และการจัดการภาพประกอบสินค้าภายในแอปพลิเคชัน

---

## 🛠️ เทคโนโลยีที่ใช้

- **Language**: C# (C-Sharp)
- **Framework**: .NET Framework / Windows Forms (WinForms)
- **Database**: Microsoft Access (`.accdb`)
- **IDE**: Microsoft Visual Studio

---

## 📂 โครงสร้างโปรเจกต์โดยย่อ

```text
mobileshop/
├── Form1.cs               # หน้าจอหลักและการจัดการสต็อกสินค้า
├── Form2.cs               # ระบบจัดการข้อมูลพนักงาน/ผู้ใช้งาน
├── Form3.cs               # ระบบขายสินค้าและสรุปยอด
├── MoblieShop.accdb       # ฐานข้อมูลหลักของระบบ (Access Database)
├── bin/debug/             # ไฟล์ Executable และ Resource สำหรับรันโปรแกรม
└── README.md              # ไฟล์อธิบายโปรเจกต์
```
